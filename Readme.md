### 🎤 Text-to-Speech using Python

A simple Python Text-to-Speech (TTS) program that converts text into spoken voice using 

- **Python**
- **pyttsx3 library**

This project demonstrates how Python can be used to create a basic voice assistant feature.

<h2><a class="anchor" id="Project Overview"></a>Project Overview</h2>

This program uses the pyttsx3 library to convert text into speech.
It initializes the speech engine, sets the voice and speaking rate, and then speaks the provided text.

---
<h2><a class="anchor" id="Features"></a>Features</h2>

- Convert text into speech
- Adjustable speech rate
- Supports different voices
- Offline speech synthesis (no internet required)


---
<h2><a class="anchor" id="Technologies Used"></a>Technologies Used</h2>

<h2><a class="anchor" id="Installation & Setup"></a>Installation & Setup</h2>

- Clone the Repository

  - git clone https://github.com/yourusername/text-to-speech-python.git

- Navigate to the project folder

  cd text-to-speech-python

- Install required libraries

  - pip install pyttsx3 
  - SpeechRecognition 
  - pyaudio

--- 
<h2><a class="anchor" id="How to Run"></a>How to Run</h2>

- Run the Python script:

  python main.py

The program will speak the text message.
